# MarketLlama Preview

MarketLlama is an AI-powered market analysis and trading interface focused on clean charting, structure analysis, and trader workflow.

## What this repo is
This is a public preview repository for MarketLlama.
It is shared only to demonstrate product direction, UI quality, and chart experience.

## What this repo does NOT include
- Production source code
- Backend services
- Private trading logic
- API keys
- Environment variables
- Infrastructure configuration

## Current product areas
- Live chart interface
- Multi-timeframe chart viewing
- Advanced chart tooling
- AI-assisted analysis workflow
- Trading dashboard UX
